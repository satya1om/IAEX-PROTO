# Continuity Simulation (Static, No Angular, No TypeScript)

This is a standalone, Hostinger-friendly static build for the "Physics of Economic Continuity" concept.
The render system is scene-driven and tuned for cinematic continuity (chaos → coherence → gravity → civilization field).

## Structure

- `index.html` — single cinematic page.
- `css/styles.css` — visual system (void background, luminous text treatment).
- `js/` — engine, scene manager, and scene renderers.
- `data/content.json` — editable text lines (no rebuild needed).
- `data/scenes.json` — scroll-to-scene timing map (no rebuild needed).

## Deploy to Hostinger (shared plan)

1. Upload the `continuity-sim/` contents to `public_html/`.
2. Open your domain.
3. Edit `data/*.json` for copy/timing updates without changing JS.
