# Economic Continuity Cinematic (New Clean Build)

This is a separate fresh build to avoid confusion with previous prototypes.

## Files

- `index.html`
- `styles.css`
- `main.js`
- `narrative.json`

## Hostinger deploy

1. Upload all files in this folder to a new directory such as `public_html/economic-continuity-cinematic/`.
2. Open: `https://yourdomain.com/economic-continuity-cinematic/`
3. Edit `narrative.json` to change text without touching JS.
