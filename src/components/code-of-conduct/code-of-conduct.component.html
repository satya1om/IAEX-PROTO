Code of Conduct

Our commitment to responsible, ethical, and sustainable manufacturing is non-negotiable. This Code outlines the minimum standards for all network partners.

Preamble

The IAEX Network is built on a foundation of trust, integrity, and a commitment to responsible and sustainable manufacturing. This Code of Conduct outlines the minimum standards required of all manufacturing partners. It is applicable to all scales of operation, from MSMEs to large-scale units. Compliance is a prerequisite for entry and continued participation in our network, reflecting our shared values and alignment with global standards like the Universal Declaration of Human Rights and the ILO Declaration.
